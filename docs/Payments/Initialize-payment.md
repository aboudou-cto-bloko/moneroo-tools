# Initialize payment

When you collect payments with Moneroo, you have many options. Here's a quick overview:

If you're building a website or application

* [**Moneroo Standard**](https://docs.moneroo.io/payments/standard-integration) **:** This is the basic, "standard" integration approach. To use this, you need to call our API from your server to generate a payment link. You then redirect your customer to this link for them to make the payment. Once the payment has been processed, we'll redirect them back to you.

Another options and integration will be available soon.
